# Korean Four-Ball Simulator

Live demo: https://chanyoungs.github.io/billiards-sim/

Browser-based Korean 사구 simulator with:

- top-down 2D planning view
- realistic 3D table view
- cue placement, aim, power, and spin controls
- shared simulation-backed trajectory preview and shot playback

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deployment

This project is configured to deploy to GitHub Pages via GitHub Actions.
